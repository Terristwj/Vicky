# Vicky

Ellipsis Tech Series 2023 Hackathon

### To run this project

First time setup: `npm install`
To run: `npm run start`

### To Edit the navigation

Visit the following codes in `src`:

1. `router > Router.tsx`
2. `components > navigation > NavTabs`
3. `pages > Layout.tsx`
